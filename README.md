        <h2>Key Terms</h2>
         <ul>
             <li><b>HTML</b>- HTML is referred to as HyperText Markup LanguageThe common markup language used to create Web pages is HTML. The structure of a Web page is described by HTML.</li>
              <li><b>Class</b>- One or more classes, separated by spaces, are present in HTML elements. By choosing elements based on their classes, you can style them with CSS.
</li>
              <li><b>ID</b>- An id attribute can be used to identify an HTML element. Each element should never contain more than one id, and id elements should always be specific to that one element.</li>
              <li><b>HRef</b>- Links use a href, which stores a URL and instructs the browser where to go.</li>
              <li><b>Body</b>- All of a page's content is contained in the body. falls inside the general <html> element, following the <head> tag.</li>
